**Learning objective:** By the end of this lesson, students will be familiar with the core service offerings of AWS, Azure, and GCP.

# Top Cloud Providers

- **Amazon Web Services (AWS):** The pioneer and market leader, known for its vast array of services.

- **Microsoft Azure:** Strong enterprise foothold, with seamless integration with Microsoft tools.

- **Google Cloud Platform (GCP):** Built on Google's massive infrastructure, with strength in big data, machine learning, and containerization.

# Scavenger Hunt

For each provider, find the equivalent service or product for the following. Note down the name of the product, and key differences or unique selling points:

- Virtual machines for running compute workloads
- Object storage for unstructured data
- Managed relational database offerings
- Networking services for connecting cloud resources
