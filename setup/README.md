## Setup

Create a new [Google Doc](https://docs.google.com/document).

Create a new [Google Slides presentation](https://docs.google.com/presentation) and title it **Your Name - Cloud Concepts and Providers Lab**, replacing **Your Name** with your name.

Publish the presentation so anyone on the internet can view it.

Save the URL of the published presentation so you're able to submit it later.
