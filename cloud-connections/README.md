<h1>
  <span class="headline">Cloud Connections</span>
  <span class="subhead">How the Cloud Connects to Your Computer</span>
</h1>

**Learning objective:** By the end of this lesson, students will have a basic understanding of how the cloud is connected to physical computers and networks.

Even though we talk about the cloud like it's up in the sky, it's actually made up of real, physical things. Every service in the cloud runs on actual computers in big buildings called data centers.
When you use a cloud service, here's what happens:

- You send a request from your computer to the cloud service over the internet.
- The request goes to the provider's data center and finds the right computer to handle it.
- That computer does the work and sends the result back to you over the internet.

So when you store a file in the cloud, it's really being stored on a hard drive or SSD in one of these data centers. And when you run a virtual machine in the cloud, it's running on a real server in a data center.

The providers take care of all the physical stuff for you - the computers, the storage drives, the network cables, the cooling systems, the power backups, and so on. That's a big part of what you're paying for when you use the cloud.
